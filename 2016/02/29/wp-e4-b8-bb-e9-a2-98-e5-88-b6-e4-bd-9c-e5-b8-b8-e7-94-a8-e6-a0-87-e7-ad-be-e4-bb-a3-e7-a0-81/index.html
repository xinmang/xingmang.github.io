WP主题制作常用标签代码

WordPress模板结构
style.css : CSS文件
index.php : 主页模板
archive.php : Archive/Category模板
404.php : Not Found 错误页模板
comments.php : 留言/回复模板
footer.php : Footer模板
header.php : Header模板
sidebar.php : 侧栏模板
page.php : 内容页(Page)模板
single.php : 内容页(Post)模板
searchform.php : 搜索表单模板

search.php : 搜索结果模板

基本条件判断
is_home() : 是否为主页
is_single() : 是否为内容页(Post)
is_page() : 是否为内容页(Page)
is_category() : 是否为Category/Archive页
is_tag() : 是否为Tag存档页
is_date() : 是否为指定日期存档页
is_year() : 是否为指定年份存档页
is_month() : 是否为指定月份存档页
is_day() : 是否为指定日存档页
is_time() : 是否为指定时间存档页
is_archive() : 是否为存档页
is_search() : 是否为搜索结果页
is_404() : 是否为 “HTTP 404: Not Found” 错误页

is_paged() : 主页/Category/Archive页是否以多页显示

    index.php常用标签
    

    调用head.php< ?php get_header();?>
调用footer.php< ?php get_footer();?>
调用sidebar.php< ?php get_sidebar();?>
调用其它文件< ?php include(TEMPLATEPATH . '/文件名'); ?>
显示注册链接< ?php wp_register(); ?>
显示登录/注销链接< ?php wp_loginout(); ?>

//首页文章调用
< ?php if(have_posts()) : ?>/*检查是否存在Post/Page*/
< ?php while(have_posts()) : the_post(); ?>/*如果存在Post/Page则予以显示 */
< ?php the_title(); ?>/*文章标题*/
< ?php the_time('Y.m.d h:i') ?>/*发表时间*/
< ?php the_category(' '); ?>/*文章分类*/
< ?php comments_popup_link('0', '1', '%', ”, 'CLOSE'); ?>/*评论数*/
< ?php edit_post_link('EDIT'); ?>/*显示编辑链接*/
< ?php the_content(''); ?>/*正文内容*/
< ?php endwhile; ?>/*While结束*/
< ?php endif; ?>/*If结束*/
